What is BERTopic?

BERTopic uses BERT sentence embeddings + UMAP dimensionality reduction + HDBSCAN clustering to discover coherent topics. It is especially powerful for short-text corpora and social media data, preserving semantic nuance lost by bag-of-words approaches like LDA.

Configuration

Python Script (run in your environment)

pip install bertopic sentence-transformers umap-learn hdbscan from bertopic import BERTopic from sentence_transformers import SentenceTransformer import pandas as pd df = pd.read_csv("commoncrawl_export.csv") docs = df["url"].tolist() # or fetched WET text embedding_model = SentenceTransformer("all-MiniLM-L6-v2") topic_model = BERTopic( embedding_model=embedding_model, min_topic_size=10, nr_topics="auto", language="english", calculate_probabilities=True, verbose=True ) topics, probs = topic_model.fit_transform(docs) df["topic_id"] = topics topic_info = topic_model.get_topic_info() print(topic_info) topic_model.visualize_topics().show() topic_model.visualize_barchart(top_n_topics=15).show() topic_model.save("berttopic_commoncrawl_model")

ðŸ“Š LDA Topic Modelling

What is LDA?

Latent Dirichlet Allocation (LDA) is a generative probabilistic model that treats documents as mixtures of latent topics. It works well with large corpora and is interpretable. Best for longer text documents like web pages fetched via Common Crawl WET files.

Configuration

Python Script (run in your environment)

pip install gensim pyLDAvis nltk pandas import pandas as pd, gensim, nltk, pyLDAvis.gensim_models from gensim import corpora from gensim.models import LdaModel from nltk.corpus import stopwords nltk.download("stopwords") df = pd.read_csv("commoncrawl_export.csv") # Fetch text via WET files for each record: # wget https://data.commoncrawl.org/{filename} stop_words = set(stopwords.words("english")) texts = [[w for w in doc.lower().split() if w not in stop_words] for doc in df["url"].tolist()] dictionary = corpora.Dictionary(texts) dictionary.filter_extremes(no_below=5, no_above=0.95) corpus = [dictionary.doc2bow(t) for t in texts] lda = LdaModel(corpus=corpus, id2word=dictionary, num_topics=20, passes=50, alpha="auto") lda.save("lda_commoncrawl.model") vis = pyLDAvis.gensim_models.prepare(lda, corpus, dictionary) pyLDAvis.save_html(vis, "lda_topics_visualization.html") print("Interactive LDA visualization saved.")
